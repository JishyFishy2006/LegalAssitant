# Processed Text Directory

This directory contains the raw text extracted from the source PDF and HTML files.

## Generation Process:
- Files are generated by the `parse_pdf.py` and `parse_html.py` scripts.
- Each source file (e.g., `no_surprises_act_2021.pdf`) will have a corresponding `.txt` file here (e.g., `no_surprises_act_2021.txt`).

## File Format:
- Plain text (.txt)
- Should retain section breaks and headings from the original document.

### Example (`no_surprises_act_2021.txt`):

```text
# No Surprises Act - Interim Final Rules

## Section 1: Background

This section provides background information on the No Surprises Act...

## Section 2: Requirements for Providers

Providers must adhere to the following requirements...
```

## Purpose:
- These files serve as the input for the record normalization step (`normalize_records.py`).
- They provide a clean, text-only version of the source material for chunking and embedding.
