# HTML Sources Directory

This directory should contain HTML files scraped from government websites.

## Expected Sources:
- **CFPB website pages** (consumerfinance.gov)
- **CMS/HHS guidance pages** (cms.gov, hhs.gov)
- **Federal agency policy pages**
- **Legal resource websites**

## Sample Files to Look For:
- Consumer protection guidance pages
- Healthcare billing information
- Patient rights documentation
- Financial services regulations

## File Structure:
HTML files should contain:
- Clear headings (h1, h2, h3 tags)
- Structured content with paragraphs
- Links to related resources
- Agency attribution

## File Naming:
- `cfpb_consumer_tools.html`
- `cms_no_surprises_guidance.html`
- `hhs_patient_rights.html`

## Content Requirements:
- Must have meaningful headings for section extraction
- Text content should be substantial (>500 words per section)
- Should include agency/source attribution
